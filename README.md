# ExpressBookReview
Express Book Review API project built using Node.js and Express for managing books and customer reviews.
# Express Book Review API

This project is a Node.js and Express based REST API for managing books and customer reviews.

## Features
- Get all books
- Get book by ISBN
- Get books by author
- Get books by title
- Add and delete reviews

## Technologies Used
- Node.js
- Express.js
- JSON Web Token (JWT)
